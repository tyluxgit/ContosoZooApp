# Vue d’ensemble du projet
Vous développez une application pour le zoo de Contoso qui coordonne les visites scolaires. Le zoo de Contoso abrite 18 espèces différentes d’animaux. Au zoo, les élèves qui visitent sont affectés à des groupes, et chaque groupe a un ensemble d’animaux qui lui est affecté. Après avoir visité leur ensemble d’animaux, les élèves changeront de groupe jusqu’à ce qu’ils aient vu tous les animaux du zoo.

---

>Par défaut, les élèves sont divisés en six groupes. Toutefois, certaines classes comptent un petit ou un grand nombre d’élèves, de sorte que le nombre de groupes doit être ajusté en conséquence. Les animaux doivent également être affectés de manière aléatoire à chaque groupe, afin que l’expérience reste unique.

---
La spécification de conception de l’application Contoso Petting Zoo est la suivante :

## Il y a actuellement trois écoles qui visitent le zoo

* L’école A compte six groupes de visiteurs (le nombre par défaut)
* L’école B a trois groupes de visiteurs
* L’école C a deux groupes de visiteurs

Pour chaque école en visite, effectuez les tâches suivantes

1. Randomiser les animaux
2. Affecter les animaux au nombre correct de groupes
3. Imprimer le nom de l’école
4. Imprimer les groupes d’animaux
